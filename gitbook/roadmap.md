# Roadmap

## Q1 2026 — Platform Launch

* FOMO Meter with dynamic scoring engine
* AI-powered signal analysis via Anthropic Claude
* Safety Radar with 5 on-chain security checks
* Trust Tier System with burn-based verification
* Launch source detection (Pump.fun, LetsBonk, Bags.fm)
* Profile claiming with image uploads
* Admin panel for platform management
* Birdeye TradingView chart integration
* Email subscriptions and receipts via Resend

* Enhanced token discovery and filtering
* Multi-chain expansion research and feasibility study
* Community voting and sentiment features
* Improved mobile-responsive experience

## Q3 2026 — Community & Tools

* Community governance features
* Enhanced watchlist and portfolio tracking
* Alert system for FOMO score changes and safety events
* API access for third-party integrations
* Advanced analytics dashboard

## Q4 2026 — Scale & Growth

* Additional blockchain support beyond Solana
* Expanded Trust Tier benefits and tier levels
* Partnership integrations with DEXs and launchpads
* Historical data analysis and trend reporting
* Platform performance optimization and scaling
